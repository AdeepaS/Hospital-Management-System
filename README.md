# Hospital Management System

## Overview
A full-stack MERN-based Hospital Management System designed for managing patients, staff, appointments, medical records, and billing. Built with scalable multi-hospital-ready architecture.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication
- Role-Based Access Control

## Features (Planned)
- Authentication & Authorization
- Staff Management
- Patient Management
- Appointment Scheduling
- Medical Records
- Billing System
- Dashboard Analytics
